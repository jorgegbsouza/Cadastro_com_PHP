# Cadastro 
Cadastro de cliente usando método CRUD

Usado:
<ul>
<li>PHP7</li>
<li>HTML</li>
<li>MaterializeCSS</li>
<li>MySQL</li>
</ul>

2 - Crie uma tabela
<img src="https://raw.githubusercontent.com/jorgehtml/Cadastro_com_PHP/master/imagens_prints/criando_tabela.png" width="1000">
